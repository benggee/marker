# 项目介绍

marker是针对MinPrinter项目开发的一个基于Flutter开发的条码管理应用，支持Android和iOS平台，可以配合MiniPrinter打印机（硬件还在制作）使用。
在app上可以生成一维条形码，使用这个条码可以检索到对应的便签，可以用于快速检索便签。
